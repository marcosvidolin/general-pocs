# Dialogflow API v2 - Update Entities

POC to update Entities

## Stack

- Dialogflow
- Python 3

## Setting up

- Virtual Environment (Creating / Activating)

```bash
virtualenv venv
```

```bash
source venv/bin/activate
```

- Install all dependencies

```bash
pip install -r requirements.txt
```
