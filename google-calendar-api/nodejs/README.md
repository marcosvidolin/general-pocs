# Calendar API

Using Google Cloud API with service account.

## Setup

```
$ npm install -g firebase-tools
```

```
$ firebase login
```

```
$ firebase init functions
```

## To Deploy

```
$ firebase deploy --only functions
```

or

```
$ npm run deploy
```

## Run Local

```
$ firebase emulators:start --only functions
```
